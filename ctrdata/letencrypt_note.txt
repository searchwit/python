copy the 4 pem file into a folder. in this case they are version 4:

-rw-r--r--. 1 root root 5534 Apr 10 12:46 fullchain4.pem
-rw-r--r--. 1 root root 3749 Apr 10 12:46 chain4.pem
-rw-r--r--. 1 root root 1785 Apr 10 12:46 cert4.pem

openssl pkcs12 -export -name watchlist -in ./fullchain4.pem -inkey ./privkey4.pem -out jetty.pkcs12

enter pasword:  august0820

this will create a new file: jetty.pkcs12
-rw-rw-r--. 1 opc  opc  4637 Apr 10 13:19 jetty.pkcs12

copy the above 5 files to my local hp laptop
run the below (need java 11)
keytool -importkeystore -destkeystore watchlist_keystore -deststoretype jks -srckeystore ./jetty.pkcs12 -srcstoretype pkcs12 -alias watchlist

this will create a watchlist_keystore file
copy the watchlist_keystore file to searchwit_keystore file

cp the keystore file to java application
cp the pem file to http servers

http conf at 
/etc/httpd/conf.d/searchwit.com.conf

SSLCertificateFile /etc/pki/tls/certs/fullchain4.pem
SSLCertificateKeyFile /etc/pki/tls/private/privkey4.pem
